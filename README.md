# counter_remember

Counter App using simple remember/rememberSaveable to manage state.

## Dev env

 - macOS 11.4 (Big Sur) x64
 - Android Studio Artic Fox Patch 2
 - Android SDK version 30
 - JDK: 11
 - Gradle: 7.0.2
 - Kotlin: 1.5.21
 - Compose: 1.0.2
